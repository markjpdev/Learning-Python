## Concepts

### print()

The most basic Python function. Whatever you put inside the parentheses gets printed to the terminal.
```python
print("Hello World!")
```

### Variables

No type declaration needed. Just assign and go.
```python
name = "Mark"
age = 30
height = 5.9
```

### input()

Takes whatever the user types and stores it as a string. Wrap with int() or float() if you need a number.
```python
name = input("Enter your name: ")
age = int(input("Enter your age: "))
```
